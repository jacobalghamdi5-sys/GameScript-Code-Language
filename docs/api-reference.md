# GameScript v1.0 Start API Reference

## Text Controls
### `subtitle.print-text:{text_payload}`
- **Description**: Renders informational text directly underneath heading panels.
- **Parameters**: `text_payload` (String encapsulation wrapped via curly braces).

## Layout Controls
### `text-box(limit; style)`
- **`limit`**: Defines constraints (e.g., `"1-10"`).
- **`style`**: Assigns predefined skins (e.g., `"Classic Modern"`).

## Style Variables
### `Wallpaper`
- **Acceptable Values**: Strict hex formatting (e.g., `"Red #FF0303"`).
